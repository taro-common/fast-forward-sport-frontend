# FAST Forward Sport Website

High-impact motorsport company website built with Next.js + TypeScript.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.

## Project Notes

- Main layout and sections live in src/app/page.tsx.
- Global styling is in src/app/globals.css.
- Image placeholders live in public/ and should be replaced with final photography.
